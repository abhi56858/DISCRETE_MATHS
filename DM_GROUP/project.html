Logical Laws Reference
Fundamental equivalences in propositional logic.
1
De Morgan's: ¬(P∧Q) ≡ ¬P∨¬Q  |  ¬(P∨Q) ≡ ¬P∧¬Q
2
Distribution: P∧(Q∨R) ≡ (P∧Q)∨(P∧R)
3
Contrapositive: (P→Q) ≡ (¬Q→¬P)
4
Absorption: P∨(P∧Q) ≡ P
5
Double Negation: ¬(¬P) ≡ P
6
Idempotent: P∧P ≡ P  |  P∨P ≡ P
Predicate Logic
Quantifiers extend propositional logic over domains.
∀x P(x) — "For all x, P(x) is true"
∃x P(x) — "There exists x such that P(x)"
¬∀x P(x) ≡ ∃x ¬P(x)
¬∃x P(x) ≡ ∀x ¬P(x)
Example: "All students pass" → ∀x (Student(x) → Pass(x))

Key Concepts

Empty Set: Set with no elements. |∅| = 0

Set Laws
A ∪ ∅ = A    A ∩ U = A
A ∪ A̅ = U    A ∩ A̅ = ∅
A ∪ B = B ∪ A   (Commutative)
(A ∪ B) ∪ C = A ∪ (B ∪ C)   (Associative)
A ∩ (B ∪ C) = (A∩B) ∪ (A∩C)   (Distributive)
(A ∪ B)̅ = Ā ∩ B̅   (De Morgan)

Graph Theory Reference
V
Vertex/Node: Fundamental unit of a graph
E
Edge: Connection between two vertices
d
Degree: Number of edges incident to a vertex. Σdeg(v) = 2|E|
K
Complete Graph Kₙ: Every pair of vertices connected. |E| = n(n-1)/2
χ
Chromatic Number: Min colors to color vertices (no two adjacent same color)
Handshaking Lemma
The sum of all vertex degrees equals twice the number of edges:
Σ deg(v) = 2|E|
Euler's Theorem:
A connected graph has an Euler circuit iff every vertex has even degree. It has an Euler path iff exactly 2 vertices have odd degree.
Planar Graph (Euler's Formula):
V − E + F = 2

Counting Principles
+
Rule of Sum: If task A can be done in m ways and B in n ways (mutually exclusive), total = m + n
×
Rule of Product: If task A can be done in m ways and B in n ways (independent), total = m × n
!
Permutation P(n,r): Ordered arrangements of r items from n: n!/(n−r)!
C
Combination C(n,r): Unordered selections of r from n: n!/[r!(n−r)!]
D
Derangement D(n): Permutations with no fixed point: n!·Σ(−1)ᵏ/k! for k=0..n
B
Binomial Theorem: (x+y)ⁿ = Σ C(n,k)xᵏyⁿ⁻ᵏ
Pigeonhole Principle
If n+1 or more objects are placed in n boxes, at least one box contains 2 or more objects.
⌈m/n⌉ objects in at least one box (m objects, n boxes)
Example: Among 13 people, at least 2 share a birth month (12 months). Among any 5 integers, at least 2 have the same remainder mod 4.

Relation Properties
R
Reflexive: (a,a) ∈ R for all a ∈ A — every element related to itself
S
Symmetric: (a,b) ∈ R ⟹ (b,a) ∈ R — matrix is symmetric
T
Transitive: (a,b) ∈ R ∧ (b,c) ∈ R ⟹ (a,c) ∈ R
A
Antisymmetric: (a,b) ∈ R ∧ (b,a) ∈ R ⟹ a = b
Special Relation Types
Equivalence = R + S + T Partial Order = R + Antisym + T Total Order = Partial + (a≤b or b≤a)
Equivalence Classes: Partition A into disjoint subsets where all elements are mutually related.
A/R = {[a] | a ∈ A} — quotient set
Closure: Reflexive, symmetric, and transitive closures extend a relation to satisfy a property with minimum additional pairs.
